## Overview

This repository contains the official landing page for **DV Consultancy**, built to the full brand specification defined in the DV Consultancy Brand Guidelines v1.0.

### Brand Colours

| Token | Hex | Usage |
|-------|-----|-------|
| Deep Navy | `#0A1628` | Primary backgrounds |
| Midnight Blue | `#12233D` | Secondary dark |
| Consultancy Gold | `#BF9B30` | Premium accent, CTAs |
| Consultancy Teal | `#005B6E` | Technology pillar |
| Off-White | `#F8F6F2` | Light section backgrounds |
| Charcoal | `#2C3E50` | Body text |

### Typography
- **Display / Headlines:** Montserrat (Bold 700–900)
- **Body / UI:** Inter (Regular 400, Medium 500)

## Features

- Fully responsive (mobile, tablet, desktop)
- Scroll-reveal animations via `IntersectionObserver`
- Animated stat counters on scroll entry
- Sticky frosted-glass navigation on scroll
- Hero parallax effect
- Mobile hamburger menu
- Brand-accurate logo SVG (two overlapping rounded squares — Blue/Gold)
- Decorative brand motif outline shapes in hero
- Photography via Unsplash CDN

## Deployment

This is a static HTML/CSS/JS site — deploy to any static host:

- **Netlify:** Drag & drop the project folder
- **Vercel:** `vercel --prod`
- **GitHub Pages:** Enable Pages on `main` branch, root `/`
- **Custom server:** Serve `index.html` from any web server
